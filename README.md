# Wumpus World game to learn Propositional logic and Predicate logic

## - install prolog
[https://www.swi-prolog.org/download/stable](https://www.swi-prolog.org/download/stable)
## - Clone
git clone https://github.com/nvdieu/wumpus-prolog.git
## - wumpus game play
cd wumpus-prolog

swipl

-? consult('wumpus.qlf').

-? start.
